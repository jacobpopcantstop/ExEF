Verify Certificate

ExEF does not run a public credential directory lookup by ID alone. Verification works from the signed token issued with the certificate. The credential ID is used as an optional cross-check.

How this verification works

  • Use the full verification link from the certificate when possible.
  • Paste the signed receipt token if you were given the token separately.
  • Enter the credential ID to confirm the token matches the displayed certificate.
