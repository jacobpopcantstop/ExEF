    Profile Frame

    Pressure-Sensitive Activation Pattern

    This sample client understands priorities but stalls when tasks carry visibility, ambiguity, or emotional risk. The startup cost is the main problem, not intelligence or commitment.

    Best leverage point: lower task threat and create a tiny visible first action before asking for full focus.

    Signal Strengths

    • Activation: high signal across task start friction and EF story patterning
    • Regulation: moderate signal when evaluation or urgency spikes
    • Planning: moderate signal tied to optimistic timing and overcommitment
    • Environment: light signal; clutter and interruptions make the startup cost worse

    Top Priorities

    • Replace giant task lists with one first move per priority
    • Use a body-double or co-working block for high-threat admin work
    • Add a time correction factor to planning so commitments stop collapsing
    • Reduce perfectionistic over-prep before visible deliverables

    7-Day Plan

    1. Choose one recurring task and define a 5-minute start ritual.
    2. Pre-stage materials the night before so initiation does not begin from zero.
    3. Use one body-doubling block for admin or reply work.
    4. Apply a 1.5x planning correction to one week of calendar estimates.
    5. Review what actually got easier, not what still feels idealized.
