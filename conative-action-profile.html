Conative Action Profile

Discover how you naturally initiate action. This assessment measures your instinctive operating mode across four dimensions — not personality or intelligence, but how you do when free to be yourself.

Back to Assessment Hub Open Cross-Signal Profile

Educational tool, not a diagnosis. This is a self-reflection tool for coaching-style insight. It does not diagnose ADHD, executive function disorders, learning disabilities, or any mental-health or medical condition. For diagnostic evaluation, work with a licensed clinician.

What This Measures

The Conative Action Profile maps four modes of taking action: how you gather information, build systems, embrace change, and work with tangible materials. Everyone uses all four — the profile shows where you lean hardest when striving.

Fact Finder — How you gather and use information.
Follow Thru — How you organize and create systems.
Quick Start — How you handle risk and change.
Implementor — How you work with tangible materials.

How To Take This

  • You will face 36 scenarios across six life domains.
  • For each, pick the action you are MOST likely to do.
  • Then, pick the action you are LEAST likely to do.
  • Trust your gut. Answer how you naturally act when free to be yourself, not how you think you should act.
  • Takes approximately 10–15 minutes.

This tool uses a forced-choice methodology to measure conative behavior. It is an independent, non-diagnostic tool and is not affiliated with any trademarked assessment product.
