ExEF Website To-Dos (Systematic Execution)

Primary objective: Launch ExEF as a founder-led coaching offer and sell the first 5 Beta Sprints.

90-day target: 5 beta clients · 20 sessions · 5 testimonials · 1 refined offer.

Status (May 1, 2026): Website implementation is about 98% complete. Core offer messaging and fit-call CTAs are live; operational docs and outreach assets remain.

Phase 1 — Launch-Critical Website Tasks (Do First)

Outreach starts immediately after this phase is complete.

  • Homepage clearly states who ExEF helps (creative adults with ADHD/EF friction).
  • Service section explains The ExEF Beta Sprint in plain language.
  • Pricing is visible as 4-session Beta Sprint: $200 (first 5 clients only).
  • Main CTA appears above fold: Book a 15-minute fit call.
  • About Jacob section is present and credibility-focused (experience + ADDCA-in-progress).
  • Scope disclaimer is visible on coaching pages.
  • Booking/contact links work on desktop + mobile.

Phase 2 — Offer Clarity and Trust

  • Add practical promise: Build systems that survive real life.
  • Add pain-point bullets (start, plan, time blindness, avoidance, follow-through, finish).
  • Add the 4-session structure block (Friction Map → System Build → Real-Life Test → Maintenance Plan).
  • Add “what this is / what this is not” language (coaching, not therapy).
  • Keep messaging package-based (never market as $50/hour).

Phase 3 — Scope, Safety, and Boundary Language

  • Publish scope statement (not therapy, diagnosis, medical, legal, or crisis support).
  • Add referral-out triggers list for out-of-scope or clinical cases.
  • Add boundary script for clinical/legal/medical questions.

Phase 4 — Conversion QA (Before Outreach)

  • Click-test every CTA and nav link.
  • Verify booking flow end-to-end (fit call selected, confirmation visible, follow-up email sent).
  • Confirm pricing and beta-cap scarcity language is clear.
  • Fix typos, broken links, mobile layout issues, and form failures.

Done Means Outreach Starts

When these are complete, stop polishing and launch:

  • Service page live
  • Beta Sprint description live
  • Scope disclaimer live
  • Booking link live
  • Intake form ready (open)
  • Coaching agreement ready (open)
  • Payment method ready (manual invoice/checkout process documented)
  • 50-name outreach list ready (template)

Week 1 Execution (Immediately After Launch)

  • Publish beta announcement content piece (draft page).
  • Send first 20 outreach messages.
  • Book at least 2 fit calls (pending real outreach results).
  • Track leads in one simple sheet (no overbuilt CRM) (template).

Anti-avoidance rule: For every content piece published, send at least 5 direct outreach messages.

Ops support: Weekly scoreboard ready (open).
